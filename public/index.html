📡 Public API Reference

PUBLIC — No authentication required
GET
/api/movies
Fetch all movies. Returns newest first.
GET
/api/movies/{id}
Fetch a single movie by its ID.
🔍 Search Query Parameters
Combine any of these on /api/movies — all are optional and can be mixed freely.
title Filter by movie title (partial, case-insensitive) /api/movies?title=inception
genre Filter by genre (partial, case-insensitive) /api/movies?genre=sci-fi
year Filter by exact release year /api/movies?year=2010
cast Search within cast/actor names (partial, case-insensitive) /api/movies?cast=dicaprio
q Global search across title, genre, cast, description, distributor /api/movies?q=nolan
combine Mix multiple filters — all must match (AND logic) /api/movies?genre=action&year=2022

PROTECTED — Requires Authorization: Bearer <password>
POST
/api/movies
Create a new movie. Body: { title, year, genre, distributor, cast, youtubeLink, thumbnail, description }
PUT
/api/movies/{id}
Update any fields of an existing movie by ID.
DELETE
/api/movies/{id}
Permanently delete a movie by ID.
RATINGS — No authentication required
GET
/api/ratings/{movieId}
Fetch aggregated rating for a movie — averageRating, totalRatings, and per-star breakdown. Also embedded inside every /api/movies response automatically.
POST
/api/ratings/{movieId}
Submit a rating. Body: { "rating": 1–5 }. No user details stored — only the star count is aggregated. Average auto-recalculates on every submission.
📊 Rating Response Format
{
  "success": true,
  "rating": {
    "movieId": "movie_1720000000_abc123",
    "averageRating": 4.3,
    "totalRatings": 15,
    "breakdown": {
      "5": 8,
      "4": 4,
      "3": 2,
      "2": 1,
      "1": 0
    },
    "lastRatedAt": "2024-07-01T12:00:00.000Z"
  }
}

{ "rating": 5 }
